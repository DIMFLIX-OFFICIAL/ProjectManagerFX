# ProjectManager
ProjectManager - удобный и интуитивно понятный графический интерфейс для управления своими проектами. Добавляйте, маркируйте, запускайте!
Поддерживаются многие IDE включая:
- IntelliJ IDEA
- Pycharm
- Rust Rover
- VsCode
- SublimeText 3
